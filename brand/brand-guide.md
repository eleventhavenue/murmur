# Murmur Brand Guide

## Brand Personality
Quiet. Intimate. Elegant. Premium.
Like a whisper made visible.

## Color Palette

### Primary
| Name | Hex | Usage |
|------|-----|-------|
| Surface | `#161618` | Backgrounds, dark UI |
| Gold | `#C8A87C` | Primary accent, CTAs, icons |
| Text Primary | `#E8E8EC` | Headings, body on dark |
| Text Secondary | `#8A8A94` | Labels, captions |

### Extended
| Name | Hex | Usage |
|------|-----|-------|
| Gold Dim | `#A08060` | Hover states, borders |
| Gold Bright | `#D4B88C` | Active states |
| Surface Hover | `#1E1E22` | Card hover, subtle lift |
| Surface Elevated | `#242428` | Modals, dropdowns |
| Error | `#EF4444` | Error states |
| Success | `#10B981` | Success states |

## Typography

### Primary: Inter
- Light (300) — wordmark, headings
- Regular (400) — body text, UI
- Medium (500) — buttons, labels

Letter spacing: `0.08em` for wordmark, `0.02em` for body

### Wordmark
- All lowercase: **murmur**
- Inter Light 300
- Letter spacing: 0.08em
- Never capitalize, never bold

## Logo Concepts

### Concept 1: The Ripple (Recommended)
Sound waves emanating from a point — the visual shape of a whisper spreading.
Best for: brand recognition, memorable, scales well.

### Concept 5: The Pulse
Audio waveform pulse — minimal, technical, modern.
Best for: developer/open-source audience, favicon clarity.

## Tagline Options
- "Your text, read beautifully."
- "Listen to anything."
- "Hear your words."
- "Text to speech, done right."

## Voice & Tone
- Conversational but not casual
- Confident but not boastful
- Helpful but not patronizing
- Technical when needed, never jargon-heavy
